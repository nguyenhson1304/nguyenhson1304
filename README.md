## 🚀 Featured Data Projects

### 1. [banking-analytics](https://github.com/nguyenhson1304/banking-analytics.git)
* **Objective:** Designed and orchestrated a fully automated ELT pipeline to ingest, clean, and model raw financial data (transactions, customers, and cards) into a dimensional data warehouse for business intelligence.
* **Tech Stack:** Python, dlt (Data Load Tool), DuckDB, dbt, Apache Airflow, Docker, Power BI.
* **Impact:** Successfully built a blazing-fast, lightweight local data stack by leveraging DuckDB and dlt. Automated strict data quality checks (e.g., handling invalid CVVs) and historical data tracking (SCD Type 2) via dbt snapshots. The modeled data was directly integrated into Power BI to serve high-performance dashboards tracking critical KPIs like transaction volume, active customers, and error rates.
